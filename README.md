# Atelier de Montres Suisses — веб-макет

## Структура файлів
```
index.html       — розмітка сторінки
style.css        — стилі (кольори, шрифти, сітка)
images/          — усі зображення (зараз плейсхолдери)
```

## Як підставити свої фото

У папці `images/` лежать плейсхолдери з підказками прямо на зображенні —
що саме там має бути. Просто заміни файл, **зберігши ту саму назву**,
і сайт підхопить нове фото автоматично.

| Файл | Де використовується | Що показати |
|---|---|---|
| `logo.png` | Header + Footer | Логотип ательє (вже фінальний, можна не змінювати) |
| `watch-01-macro-dial.jpg` | Hero, Tiers, Portfolio | Макро циферблата (гравіювання / розпис) |
| `watch-02-movement.jpg` | Hero, Bespoke Service, Journey, Portfolio | Скелетон-механізм крупним планом |
| `watch-03-on-wrist.jpg` | Hero, Portfolio, Contact | Готовий годинник на запʼясті з ремінцем |
| `watch-04-case-detail.jpg` | Hero, Tiers, Portfolio | Деталь корпусу або корони |
| `watch-05-side-profile.jpg` | Hero, Tiers, Portfolio | Бічний профіль корпусу |
| `watch-06-finished-piece.jpg` | Hero, Tiers, Portfolio | Готовий виріб, загальний план |
| `editorial-01-workshop.jpg` | Portfolio (бічна колонка) | Атмосфера ательє, вертикальне фото |
| `editorial-02-craft.jpg` | Bespoke Service (бічна колонка) | Процес ручної роботи, вертикальне фото |

**Орієнтовні розміри:**
- квадратні фото годинників (`watch-*`) — оптимально 1200×1200px і більше, формат JPG
- вертикальні editorial-фото — оптимально 1200×2100px (співвідношення ~3:4 і вужче)
- логотип — вже готовий PNG з прозорим/власним фоном

Кожне фото в HTML підписане атрибутом `alt`, що теж описує бажаний кадр —
зручно орієнтуватись, відкривши `index.html` у текстовому редакторі
та шукаючи потрібну назву файлу (Ctrl+F).

## Технічні примітки
- Чиста статика: один `index.html` + один `style.css`, без збірки і фреймворків.
- Шрифти підключені з Google Fonts (Prata, Cormorant Garamond, Raleway, Jost) —
  для повністю офлайн-роботи їх можна завантажити локально і прописати `@font-face`.
- Адаптивна верстка: перевірена на desktop (1440px) і mobile (390px).
